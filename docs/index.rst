Welcome to Ulmo's documentation!
============================

**Ulmo** is a deep learning framework for anomaly detection in oceanographic data, leveraging probabilistic autoencoders and normalizing flows for identifying unusual or out-of-distribution phenomena in sea surface temperature and other oceanographic fields.
