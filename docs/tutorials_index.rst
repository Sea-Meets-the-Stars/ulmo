Tutorials
=========

This section contains step-by-step tutorials for using Ulmo to analyze oceanographic data.

Basic Usage Tutorial
------------------

The :doc:`basic_usage` tutorial introduces the fundamental concepts of Ulmo and demonstrates how to load pre-trained models and evaluate sample data.

Data Preprocessing Tutorial
------------------------

The :doc:`data_preprocessing` tutorial covers how to prepare raw oceanographic data for analysis with Ulmo, including handling cloud masks, extracting cutouts, and applying preprocessing steps.

Model Evaluation Tutorial
----------------------

The :doc:`model_evaluation` tutorial explains how to evaluate oceanographic data using pre-trained models and interpret the results, including log-likelihood scores and reconstructions.

Anomaly Detection Tutorial
-----------------------

The :doc:`anomaly_detection` tutorial demonstrates how to identify and analyze anomalous patterns in ocean temperature data using Ulmo's probabilistic framework.

Custom Models Tutorial
-------------------

The :doc:`custom_models` tutorial shows how to create, train, and evaluate custom models for specific oceanographic applications.

Spatial Analysis Tutorial
----------------------

The :doc:`spatial_analysis` tutorial covers techniques for analyzing and visualizing the spatial distribution of anomalies and ocean patterns.
